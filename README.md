# Restaurant Page

Simple restaurant home page.

## How to run the code

- Clone the file to your computer

```bash
git clone https://github.com/prefolo/restauraunt-page.git
```

- Enter the repository directory

```bash
cd restauraunt-page
```

- Install dependencies:

```bash
npm install
```

- Build the bundle:

```bash
npm run build
```

- Open the site:

```bash
npm run start
```
